# Fragrance Catalog
A web app to browse, search, and review fragrances.

## How to run application:

### 1. Setup backend
npm install
docker-compose up
npm start

### 2. Setup frontend
npm install
npm run dev

### 3. Open website
The app runs on `http://localhost:5173`

## Features
- User authentication (signup/login)
- Browse fragrance catalog
- Search by name or brand
- Filter by notes (top, middle, base)
- Sort by popularity, rating, or price
- Add, edit, and delete fragrances
- Add, edit, and delete personal reviews
- Update pricing information
